## props 传值

```jsx
function Component() {
  const p = {
    name: 'jack',
    age: 18,
    sex: 1
  }
  return (
    <Person name={p.name} age={p.age} sex={p.sex}></Person>
  )
  
  // 等价于
  return (
    <Person {...p}></Person>
  )
}
```

在react jsx模板语法中外层的花括号并不代表对象的花括号，`{}`表示js脚本的编写范围，同时我们知道扩展运算符`...`并不能直接作用于对象，这里可以当做一个语法糖来处理
